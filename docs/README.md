### 🔍 **Quick Diagnostic**

**Package not working?**
1. Check [Troubleshooting - Package Not Working](TROUBLESHOOTING.md#package-not-working--tests-not-being-captured)
2. Enable debug mode: `--vars '{dbt_test_results: {debug_mode: true}}'`
3. Verify configuration in [Getting Started](GETTING_STARTED.md#configuration-deep-dive)

**Performance issues?**
1. Check current scale in [Performance Benchmarks](PERFORMANCE_BENCHMARKS.md#detailed-performance-matrix)
2. Optimize configuration using [Performance Guide](PERFORMANCE_BENCHMARKS.md#configuration-optimization-by-scale)
3. Monitor with queries from [Performance Monitoring](PERFORMANCE_BENCHMARKS.md#performance-monitoring)
